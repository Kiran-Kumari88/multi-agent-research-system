# 🔬 ResearchMind – Multi-Agent AI Research System

ResearchMind is a Multi-Agent AI application that automatically researches any topic using specialized AI agents. It searches the web, extracts relevant information, writes a structured research report, critiques the report, and allows downloading it as Markdown or PDF.

## 🚀 Live Demo
<https://multi-agent-research-system-tsgna8pwh8spv4djcqzabb.streamlit.app/>

## ✨ Features

- Multi-Agent Architecture
- Web Search Agent
- Reader/Scraper Agent
- AI Research Writer
- AI Critic & Report Evaluation
- Professional Streamlit UI
- Markdown Report Download
- PDF Report Download
- Responsive Interface

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- LangGraph
- Mistral AI
- Tavily Search API
- BeautifulSoup
- ReportLab
